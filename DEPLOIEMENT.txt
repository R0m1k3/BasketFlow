========================================
  DÉPLOIEMENT BASKET FLOW - 100% AUTOMATIQUE
========================================

Sur votre serveur privé, dans le dossier du projet, EXÉCUTEZ CES 3 COMMANDES :

docker-compose down
docker-compose build --no-cache
docker-compose up -d

C'EST TOUT ! Allez sur votre URL et tout fonctionnera automatiquement.

========================================
  CE QUI SE PASSE AUTOMATIQUEMENT
========================================

1. Au chargement de la page :
   → L'app détecte les tokens invalides
   → Les supprime automatiquement
   → Recharge la page automatiquement
   
2. Page de connexion affichée :
   → Identifiant : admin
   → Mot de passe : admin

3. Une fois connecté :
   → Administration → Utilisateurs : FONCTIONNE ✅
   → Administration → Configuration API : FONCTIONNE ✅
   → Administration → Logos : FONCTIONNE ✅
